# BBS Memo (仮称) — 落ちる掲示板風メモ帳

「放置するとスレッドが落ちる」という2ちゃんねる（現5ちゃんねる）型掲示板の文化をメモ・リマインダーに応用したサーバーレスSPAアプリケーション。

## 🌟 主な特徴

1. **スレ落ちリマインダー**:
   - スレッド（タスクやメモ）ごとに落ち時間（5分〜30日）を設定可能。
   - 最終書き込みから設定時間が経過、または最大レス数（1000レス）に達するとスレが落ち、閲覧専用の「過去ログ（アーカイブ）」に自動移行します。
   - スレを維持（保守）するためには、能動的にメモ（レス）を書き込む（または保守ボタンを押す）必要があります。

2. **次スレ（Part 2）自動作成**:
   - 1000レスに達してスレが落ちた際、元の設定（落ち時間、AI設定、タグなど）を引き継ぎ、タイトル末尾のナンバリングを自動でカウントアップ（例: 「勉強メモ」→「勉強メモ 2」）した新スレをワンタップで作成できます。

3. **マルチプロバイダ対応 AI 自動レスポンス**:
   - Google Gemini、Anthropic Claude、OpenAI、および各種 OpenAI互換API（DeepSeek、Groq、Ollamaなど）に対応。
   - スレッド履歴全体をコンテキストにした自動書き込み（AIによる返信・突っ込み）を返信間隔タイマー（数分〜数日、ランダム）に応じてバックグラウンドで実行。
   - 動的に利用可能なモデル一覧を取得し、ドロップダウンから選択できる機能も搭載。

4. **アタッチメントと画像ユーティリティ**:
   - 1レスあたり最大4ファイル（合計15MB）までのファイルを添付し、IndexedDBへBase64保存。
   - Canvas APIを使用した端末内画像圧縮（画質調整・プレビュー付き）。
   - 画像をテキスト等幅フォントに適したアスキーアート（AA）に変換し、本文に挿入できるAA変換機能。

5. **高度な検索とフィルタリング**:
   - 全文検索、スレタイ検索、スレ内検索、ファイル名検索、投稿者名検索、ID検索に対応。
   - 任意のワードやIDによるNG（あぼーん）表示フィルター機能。

6. **ローカル完結型セキュリティ**:
   - 完全なサーバーレス設計。APIキーやファイル等のデータはすべてブラウザ内の `IndexedDB` にローカル保存され、外部に送信されません。
   - JSZipライブラリを利用した全データ（添付ファイル含む）のZIP形式インポート・エクスポート。
   - ストレージ使用量ダッシュボードによる容量警告。

## 🛠️ 技術スタック

* **コア**: HTML5 / JavaScript (Vanilla ES6)
* **スタイル**: CSS3 (カスタムプロパティによるマルチテーマ、レスポンシブ対応)
* **ストレージ**: IndexedDB (BBSMemoDB)
* **外部ライブラリ**: JSZip (ZIPデータ入出力用、CDN経由)、FontAwesome (アイコン用)

## 🚀 使い方

### 起動方法
本アプリケーションはビルドが不要な静的Webページです。
1. `index.html` を任意のウェブブラウザでダブルクリックして開く。
2. または、任意のローカルHTTPサーバー（VS Codeの `Live Server`、Python の `python -m http.server` など）を利用してホストし、アクセスする。

### APIキーの設定
AIレス機能を使用する場合は、画面右上の「ギア（設定）」アイコンまたは下部ナビゲーションの「設定」から設定画面を開き、利用したいAIプロバイダのAPIキーおよびモデルを設定してください。
※ OpenAI互換API（LM StudioやOllamaなど）をご利用の場合は、カスタムエンドポイント欄にAPIベースURLを正しく指定してください。
